CherryLib
=========

Cherry Library - Win32 Unified Library

Win32/MFC 기반의 Application을 개발하기 위해 필수적인 기능들을 모은 유용한 라이브러리입니다. 
이 프로젝트는 Visual Studio 2013에서 개발되었습니다.

1. CherryCtrl: GDI+ 기반으로 제작된 UI 컨트롤 DLL 라이브러리

  - Cherry 향나는 예쁘고 말랑한 느낌의 UI Control을 구현하자는 마음에서 시작함.
  - 장점: 
    1. MFC와 GDI+를 사용하여 이미지 파일 포맷 사용의 다양성, 투명 이미지를 지원하여 깔끔한 UI 구현을 할 수 있다.
    2. 다중 상속을 활용하여 코드 재사용성을 높임.
    3. 굳이 third-party 라이브러리를 사용하는 것보다는 자체적인 MFC 컨트롤 기반의 이질감 없는 UI 컨트롤 사용 수요도 있을 것
    이기에 Windows 개발자들에게는 굉장히 유용한 라이브러리가 되리라 믿어 의심치 않습니다! :D

  - 단점: 
    1. GDI+ 때문에 느리다.
    2. 크로스플랫폼 따위는 개나.... =_=a
  
  - CherryExam 프로젝트 예제 참조.

2. CherryUpdate: 라이브 업데이트 DLL

3. CherryUtil: 유틸리티성 함수들을 모아놓은 DLL 라이브러리

짬짬히 공들여서 개발한 라이브러리입니다. 아직 뭐 이렇다할 문서도 없고 현재 진행형인 프로젝트이지만 공개해봅니다.
